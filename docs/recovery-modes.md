# Recovery Modes

PowerRecover has several ways to search for files.

## Recommended Search

Best first choice. It tries to find useful files while filtering junk, system
noise, and unreadable matches.

## Folder Check

Use this when you want to inspect a normal folder and copy good files with
preview support. This mode does not recover deleted files.

## Deleted File Recovery

Use this for deleted files on a drive. Run as Administrator and save recovered
files to a different drive.

## Raw File Search

Use this when the file system is damaged or the drive appears as RAW. It searches
for known file signatures. Raw search may recover files without original names
or folders.

## Disk Image Search

Use this when you have a backup, clone, or raw disk image. This is safer than
scanning a failing physical drive directly.

## File Quality

The quality score estimates whether a found file looks readable. A higher
minimum quality reduces junk but may hide damaged files that are still partly
useful.
