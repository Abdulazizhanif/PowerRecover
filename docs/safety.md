# Recovery Safety Guide

File recovery is safest when you avoid changing the drive that lost data.

## Golden Rules

1. Stop using the affected drive as soon as possible.
2. Do not install PowerRecover onto the drive you are trying to recover from.
3. Save recovered files to another drive or external USB disk.
4. If the drive is physically failing, clone it first and scan the clone.
5. Preview files before recovering large batches.

## Why Recovery Can Fail

Deleted files can be recovered only if the file data has not been overwritten.
If Windows, another app, or the recovery tool writes new data to the same drive,
the old content may be permanently replaced.

## Healthy Drive

Use the normal GUI. Choose the source drive or folder, choose an output folder
on another drive, then scan.

## Failing Drive

If the drive clicks, disappears, freezes, or reports many read errors, stop
using it. Clone the drive with a dedicated imaging tool, then scan the image or
clone.

## Bootable USB

A bootable USB is useful when the damaged drive is also the Windows system
drive. Booting from USB avoids writing more data to the internal drive.
